# Critic Note: Design Options

- what_was_reviewed: `outcome/anime-streaming-landscape/options.html`
- critique_mode: `dual`
- signoff_state: `pass with cuts`

## Critique Findings

### Option A: Shelf Wall

- Strong focal point and best match for the "media shelf" framing
- Risk: broad platform and era patterns may become too hidden if summary cues are not built into the filter rail or card metadata

### Option B: Signal Table

- Defect class: `default-dashboard-grammar`
- The dense row treatment is useful, but it drifts toward a generic utility interface rather than the more inviting browseable shelf promised by the brief
- It flattens hierarchy and makes the title-level discovery experience feel secondary

### Option C: Library Desk

- Strongest balance between browsing and reading
- Best alignment with the brief's request to keep selected title details close to poster, synopsis, and platform availability
- Safer than Option A if we want visible caveats and context without turning the page into a report

## Keep

- Option A as the most visual path
- Option C as the most balanced and likely strongest default

## Cut

- Treat Option B as the weakest fit for the chosen posture unless the user explicitly wants a more analytical tool

## Redesign

- For Option A, add a visible pattern layer such as compact summary chips, availability counts, or a small overview strip so browsing does not become purely poster-driven
- For Option C, keep the left list compact but visually rich enough that it still feels like a shelf rather than a bland sidebar
- In either surviving option, place the caveat note near the filter or detail region so the warning travels with the interaction
