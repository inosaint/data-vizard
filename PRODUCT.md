# Product

## Register

product

## Users

Data Vizard is for workshop participants and agent-assisted data storytellers who need to move from a dataset to an evidence-backed HTML visualization. Users are usually comparing data fitness, analytical directions, story framing, and design choices while keeping decisions explicit.

## Product Purpose

Data Vizard provides a staged workflow for data visualization projects: data intake and curation, exploratory analysis, narrative structure, visual design, and final HTML artifact production. Success means the user can trace each visualization claim back to curated data, documented caveats, and visible design decisions.

## Brand Personality

Clear, rigorous, collaborative.

## Anti-references

- Hidden autopilot that silently chooses datasets, story frames, or chart forms.
- Decorative dashboards that obscure missingness, uncertainty, or caveats.
- Visualization artifacts that look finished but lack interaction states, verification notes, or accessible reading paths.

## Design Principles

- Make decision gates visible and lightweight.
- Keep source data, analysis, narrative, and design handoffs auditable.
- Treat caveats as part of the interface, not appendix material.
- Prefer familiar, scannable product UI for repeated comparison tasks.
- Build final HTML artifacts that can stand alone without remote dependencies.

## Accessibility & Inclusion

Prioritize readable labels, keyboard-visible focus, hover and focus equivalents where practical, reduced-motion support, and non-color cues for important meaning. Visualization copy should plainly distinguish context from cause and missing values from zero values.
