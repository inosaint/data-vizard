Data Vizard

Agent workflow plugin

Curate, shape, and ship your data’s story.

Data Vizard turns a dataset into a staged workflow. Run the stages all at once or one by one: curate the source, find the shape of the story, tighten the narrative, critique the draft, and ship a self-contained HTML outcome.

Quick start

npx data-vizard install

Use npx data-vizard install, then open a fresh Codex, Claude Code, or Gemini session so the new skills load cleanly.

Five roles, one workflow

Run them all together or individually. Each role carries a different piece of the work, so the process stays visible instead of disappearing into one opaque prompt.

database
Curator Prepares the dataset and verifies the source trustworthiness. The curator holds the system together.
query_stats
Analyst Looks for patterns, comparisons, and the shape hiding in the data.
edit_note
Narrator Tightens the narrative so the story says only what it needs to say.
verified
Critic Checks the draft for weak claims, missing links, and fuzzy handoffs.
palette
Designer Turns the finished story into a polished, self-contained HTML outcome.
