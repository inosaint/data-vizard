# Data Vizard

Data Vizard bundles agent skills for staged data visualization work: dataset intake, curation, exploratory analysis, adaptive narration, critique, design direction, and HTML visualization output.

Prefer the `data-vizard` skill for end-to-end visualization projects. Use the narrower `data-curator`, `data-analyst`, `narrator`, `critic`, or `designer` skills when the user asks for one stage directly.
